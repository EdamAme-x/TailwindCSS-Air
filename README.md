# TailwindCSS-Air
軽量版 TailwindCSSです。 エラーメッセージ等も出ない様になっています。開発ではCDN、本番ではこれを推奨します。
動作確認はしています。

## パフォーマンス

| ---- | ---- | ---- | ---- |
| CDN | 362745 bytes | [https://cdn.tailwindcss.com/3.3.3](https://cdn.tailwindcss.com/3.3.3) | 100% |
